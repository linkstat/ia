# 🤖 Inteligencia Artificial – Trabajos Prácticos

¡Bienvenid@ al repositorio de **Inteligencia Artificial**!
Aquí voy subiendo el material correspondiente a los **Trabajos Prácticos (TPs)** de la materia, como puedan ser: código, informes, documentación y cualquier otra cosa.

---

## 📁 Estructura del repositorio

```
📂 TP3/
📂 TP4/
📄 README.md
```

Cada carpeta corresponde a un trabajo práctico individual. Se espera, el docx, el PDF y los códigos:
0
* `programa.java` código fuente del TP.
* `APELLIDO-NOMBRES-TPn.pdf`: el entregable del TP.
* `APELLIDO-NOMBRES-TPn.docx`: el editable del TP entregable.
* Otros recursos (datasets, notebooks, scripts auxiliares, etc.).

Desarrollar y aplicar técnicas de Inteligencia Artificial utilizando algoritmos, herramientas y metodologías vistas en clase.
Este repositorio sirve como espacio de trabajo y seguimiento de avances.
